What is Server Side Request Forgery (SSRF)

Server-Side Request Forgery (SSRF) is a security vulnerability that allows an attacker to make requests to internal or external resources from the server where the vulnerable application is running.
This can lead to unauthorized access to internal systems, data leaks, and other malicious activities.

How SSRF Works

In an SSRF attack, the attacker tricks the server into sending a crafted request to an unintended destination.
This is typically done by manipulating input parameters that the server uses to make HTTP or other protocol requests.
